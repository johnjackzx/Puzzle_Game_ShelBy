# Shelby Slide Puzzle

A decentralized sliding tile puzzle game built on **Aptos** with fast image storage powered by **[Shelby.xyz](https://shelby.xyz/)** — Web3's first cloud-grade decentralized hot storage protocol.

Play the classic 15-puzzle (4x4 sliding tiles) where puzzle images load at Web2 speed thanks to Shelby's sub-second global reads, while game logic, scores, and economy live fully on-chain.

## ✨ Features

- **On-chain game state** — Board configuration, move counter, win detection, and high scores stored on Aptos using Move smart contracts.
- **Fast puzzle images** — Puzzle artwork stored on Shelby for low-latency, verifiable retrieval with S3-compatible access.
- **Play-to-earn / Pay-to-play** — Small entry fee in APT (or custom token) to start a game.
- **High scores** — Persistent leaderboards recorded on-chain.
- **Optional NFT rewards** — Mint a badge NFT upon solving difficult puzzles (expandable).
- **Multiple themed puzzles** — Easily add new images via Shelby uploads.
- **Wallet integration** — Connect with Petra, Nightly, or any Aptos-compatible wallet.

## 🏗️ Tech Stack

- **Blockchain**: Aptos (Move language)
- **Storage**: [Shelby.xyz](https://shelby.xyz/) — Decentralized hot storage (co-built by Aptos Labs & Jump Crypto)
- **Frontend**: React + TypeScript + Aptos SDK
- **Wallet**: Aptos Wallet Adapter
